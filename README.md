# range-value-broker

Check in and out range values as mutexes.

| Environment Variable              | Example       |
| --------------------------------- | ------------- |
| RANGE_VALUE_BROKER_RANGE          | ["1","2","3"] |
| RANGE_VALUE_BROKER_REDIS_ADDRESS  | :6379         |
| RANGE_VALUE_BROKER_REDIS_PASSWORD | Password123   |
